About the API

The goal of the Crypto Market Monitor Project is to make market data more accessible. The API is public and streamed via Socket.io.

See the Broadcast Server source on github.

See also the market-monitor npm module.

Getting Started

Base URL

http://api.marketmonitor.io

Install Socket.io

Using npm: 

$ npm install socket.io-client --save

Using bower:

$ bower install socket.io-client --save

Or load from the cdn:

<script src="https://cdn.socket.io/socket.io-1.1.0.js"></script>

Connect to a Channel

For example, to connect to the trades channel:

var trade = io('http://api.marketmonitor.io:80/BTC/USD/trades');
trade.on('trade', function(trade) {
   console.log('Trade:', trade);
});

Trades

Emits a trade event whenever a trade occurs and passes the associated trade data.
/BTC/USD/trades
Event: 'trade'
Data Format: 

{
    exchange: String,
    date: Date,
    price: Number,
    amount: Number,
    currency: 'BTC',
    tCurrency: 'USD',
    exchangeTradeID: String
}

Summary Statistics

Emits an object containing market summary statistics.
/BTC/USD/summary
Event: 'update'
Data Format: 

{
    vwap: Number,
    volume: Number,
    high: Number,
    low: Number,
    variance: Number,
    standardDeviation: Number,
    coefficientOfVariation: Number,
    range: Number,
    numTrades: Number
}

Price Chart Data

Emits an array of datapoints useful for making price over time charts.
/BTC/USD/priceCharts/:timeframe
Where timeframe is the timeframe of each datapoint.
Options:
oneMinute
fiveMinutes
fifteenMinutes
oneHour
Event: 'update'
Data Format: 

[
    {
        date; Date,
        high: Number,
        low: Number,
        open: Number,
        close: Number,
        vwap: Number,
        volume: Number,
        numTrades: Number
    },
    ...
]

Price Distribution Data

Emits an array of datapoints useful for making price distribution charts
/BTC/USD/priceDistribution
Event: 'update'
Data Format: 

[
    {
        exchange: String,
        price: Number,
        volume: Number
    },
    ...
